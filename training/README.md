# Coins
## coins repo for CSSE

### 模型介绍
- 模型结构保存在`model.py`中。
- 配置`train.py`中`dataset`的文件路径（csv文件和图片路径），直接运行该文件即可开始训练；`test.py`的使用同理。

### 文件介绍
- **kinds目录**：即为四大分类，对应类别与ID关系为：`{'XF': 0, 'VF': 1, 'AU': 2, 'MS': 3}`。
- **scores目录**：即为分数分类，对应类别与ID关系为：
```python
{
'01':0,'02':1,'03':2,'04':3, 
'06':4,'07':5,'09':6,'30':7,'35':8,'40':9,
'45':10,'50':11,'53':12,'55':13,'58':14,'60':15,'61':16,
'62':17,'62+':18,'63':19,'63+':20,'64':21,'64+':22,'65':23,'65+':24,'66':25
}
```

- `kinds`和`scores`目录下的`model`文件夹：为此前训练的结果及数据增强操作文件。
- `data.csv`：为该任务中使用的数据集，通过`split.py`文件划分`train.csv`、`test.csv`、`val.csv`。

### 数据集
- `kinds`使用的数据集名称为`now.zip`。
- `scores`使用的数据集名称为`new.zip`。
- `coins.csv`包含所有爬取的数据信息，适用于`now.zip`和`new.zip`。
