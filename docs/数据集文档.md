## 四、模型架构

### VGG19 + SE（Squeeze-and-Excitation）

```
                    正面图片(224×224×3)              反面图片(224×224×3)
                           │                                │
                    VGG19 Backbone                    VGG19 Backbone
                    + SE Attention                    + SE Attention
                           │                                │
                    特征图(512×7×7)                  特征图(512×7×7)
                           │                                │
                           └────────────┬───────────────────┘
                                        │
                                Concat 拼接
                              (1024×7×7 flattened)
                                        │
                              FC(1024×7×7×2 → 512)
                                        │
                                  ReLU + Dropout
                                        │
                                 FC(512 → 128)
                                        │
                                  ReLU + Dropout
                                        │
                              FC(128 → num_classes)
                                        │
                                    输出概率
```

**关键设计点：**
- 双路输入：正反面各走一个 VGG19 backbone，特征图拼接后分类
- SE 模块：在 5 个 maxpooling 层后插入通道注意力，增强特征表达
- 预训练权重：ImageNet1K_V1
- 数据增强：随机旋转 ±15°

## 八、已知局限性

1. **类别不均衡**：VF 类仅占 0.9%，模型对该类判断能力较弱
2. **图片未包含**：图片文件存储于训练服务器，当前仓库仅有 CSV 索引
3. **年份覆盖**：主要覆盖 1900-1949 年，清代以前和 1950 年后样本少
4. **品种集中**：Top5 品种占总量 64%，长尾品种样本不足
